# RPiHW_RaymondHughes

## Temp/Solar Monitor Project
My Project Files for this can be found in ./temp_solar_monitor/
Usage:
	main.sh
		for temperature monitor:
			-C for celsius
			-F for farenheit
		for solar monitor:
			-S
## Temperature Response (Cooling Curves)
The data taking tool is get_temp_data.sh, must be run with no inputs, will explain what to do
To view results, use make_temp_plot.py

## Solar FFT
The data taking tool is get_solar_data.sh, must be run with no inputs, will explain what to do
To view results, use make_solar_plot.py, unfortunately I was not able to get the data to show, probably because the flash was too short for the sample rate :/
